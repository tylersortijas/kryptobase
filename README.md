# Getting Started
```
git clone https://github.com/tylersortijas/kryptobase.git
cd kryptobase
npm install
npm start
```

A crypto market watcher using a Coin API. It's fully responsive
via Tailwind and utilizes over 50+ coin data that is being updated
as we speak.

# Built-With:
- HTML
- CSS
- REACTJS
- REACT Router
- Coin API
- Tailwind
